Privacy Policy

Fajar · Last updated May 2026

This document explains what information Fajar accesses, how it is used, and what it does not do. Fajar uses the device's GPS solely to calculate accurate prayer times for the user's location. No data is stored on any server, and nothing is shared with any third party.

Location

Fajar requests permission to access the device's location because prayer times are determined by geographical coordinates. The app cannot function accurately without this information.

When the app is opened, the device's coordinates are transmitted directly to the Aladhan API at aladhan.com, which returns prayer times calculated for that location. No copy of the user's location is retained by Fajar at any point in this process.

Location data is never stored on any server operated by Fajar. It is never sold, never used for advertising, and never passed to data brokers or any other third party beyond the prayer time calculation described above.

Third-Party Services

Fajar uses two external APIs to deliver its core functionality. When the device contacts these services, the device's IP address may be visible to them as a standard part of how internet requests work. No personally identifying information is transmitted by Fajar to either service.

The Aladhan API at aladhan.com is responsible for prayer time calculations. The AlQuran Cloud API at alquran.cloud provides the daily Quranic verse displayed on the home screen. No location data is included in that request. Both services maintain their own privacy policies available on their respective websites.

Data Collection

Fajar has no user accounts, no login system, and no server infrastructure of its own. The app does not collect names, email addresses, usage statistics, crash reports, advertising identifiers, browsing history, or payment information. Fajar is entirely free with no in-app purchases or subscriptions.

Device Storage

All user preferences configured within Fajar, including calculation method, notification settings, and display options, are stored locally on the device. This data does not leave the device and is not accessible to Fajar or any third party.

Children

Fajar does not collect personal information from any user, regardless of age. The app is suitable for all ages and does not knowingly gather data from children.

Changes to This Policy

If this privacy policy is updated in a meaningful way, the revised version will be published at this same URL with an updated date at the top. Continued use of the app following any update constitutes acceptance of the revised policy.

Contact

Questions or concerns regarding this privacy policy can be directed to fajar.prayertimes@gmail.com.

Fajar is built with the sole intention of serving the Muslim community. Jazakum Allahu subhanahu wata'ala Khayran.
